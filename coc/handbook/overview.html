概要¶

PyLadiesについて¶

PyLadiesは国際的なメンターシップ・グループです。私たちは、Pythonのオープンソースコミュニティ内でより多くの女性たちが積極的な参加者・リーダーになるためのサポートに重点的に取り組んでいます。私たちのミッションは、多様なPythonコミュニティを盛り上げ、育て、より発展させていくことです。そのためにアウトリーチや教育、カンファレンス、イベント、懇親会などの活動を行なっています。

またPyLadiesは、女性のためのフレンドリーなサポートネットワークを築き、より広いPython Worldへの架け橋となることも目的としています。Pythonに興味があるなら是非参加してみましょう!

このキットの使い方¶

私たちのモデル（リポジトリ）をコピーしてください（フォーク＆コントリビュート）。またはフォークして完全に違うものにします。あなたが工夫して使っても私たちは気にしません！

目的¶

これらはこのPyLadiesキットのいくつかの目的です：

  1. 自分の地域のPyLadiesを開始することに興味のある人を支援する
  2. 地域の主催者のためのガイドとなること
  3. 会員のためのガイド

現在のところ、pipで機能限定のpyladiesをインストールすることができます。（pip install pyladiesを参照）しかし、私たちはPyLadiesパッケージのためのより素晴らしいツールを計画しています。

pipでpyladiesをインストールする¶

インストール¶

pyladiesは以下を使って簡単にインストールできます：

$ pip install pyladies

使用法¶

PyLadiesのドキュメントを表示するには、次のコマンドを実行します。

$ pyladies handbook

かなり限定された機能です。しかし、この後も楽しんでください。
